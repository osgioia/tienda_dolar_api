Ejercicio 1 - API REST

Tecnología a utilizar

  • Node en cualquier versión superior a la 10.x
  • Amazon S3 y aws-sdk
  • Express
  • Cualquier librería que considere necesaria

Problema

Zeep Xanflorp, inhabitante del Miniverso, le solicita la realización de una API REST que sirva para su próximo proyecto, un e-commerce. Lo único que posee es una base de datos en formato JSON con una serie de productos que ofrece generalmente.

Acciones pedidas:

  • Obtener el producto más caro
  • Obtener los 5 productos más baratos
  • Obtener la cantidad de productos por tipo
  • Realizar una compra de un producto
    • Si hay stock, permitir la compra y restar la cantidad comprada
    • Si no hay stock, se deberá arrojar un error acorde
  • Búsqueda de productos por nombre

Adicionales

De manera adicional, se podrán desarrollar cualquiera de los siguientes puntos:

Adicional 1 - Compra fortuita

A los inhabitantes del microverso les gusta mucho los juegos de azar. Es por esto que se solicita que cuando alguien realice una compra de cualquier producto con un stock superior a 3 unidades, tenga una determinada chance de ganar. Además, la probabilidad de ganar debe aumentar conforme aumenta la cantidad de productos que compra un cliente; Es decir, si una persona compra 10 productos tiene que tener más chances de ganar que aquella que compra una cantidad inferior.

Adicional 2 - Recomendaciones

Recomendar productos a los clientes en función de aquellos que llevan comprados.

Adicional 3 - Creación de producto

Permitir el agregado de productos al sistema.
